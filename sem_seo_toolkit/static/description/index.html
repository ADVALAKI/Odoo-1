Description

Set of tools to help with Search Engine Marketing / Optimisation

*NOTE* Selenium, Google Chrome headless and the Chrome driver will need to be installed for the following SEO checks
  1. Page Load Time
  2. Non Optimised Images
If not installed the checks will be omitted from the report
https://selenium-python.readthedocs.io/installation.html
https://tecadmin.net/setup-selenium-chromedriver-on-ubuntu/
Ubuntu 64 Bit instructions
sudo pip3 install selenium
sudo su -
curl -sS -o - https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add
echo "deb [arch=amd64]  http://dl.google.com/linux/chrome/deb/ stable main" >> /etc/apt/sources.list.d/google-chrome.list
apt-get -y update
apt-get -y install google-chrome-stable
wget https://chromedriver.storage.googleapis.com/2.41/chromedriver_linux64.zip
apt install unzip
chown root:root /usr/bin/chromedriver
chmod +x /usr/bin/chromedriver
Then restart your Odoo instance
*NOTE* You will need to enter an API key and search engine ID for Google Custom Search Engine in settings for the following checks to work
  1. Google Domain Indexed
If not setup the checks will be omitted from the report
It is also needed for the SEO Competitor Analyse Report

Modular SEO checks that can be deactivated and extended
