Description

Assigns top level domains to database of any name

Instructions

You need to add an option to your odoo/tools.config.py file specifing the name of the database acting as the saas server
group.add_option("--sythilsaasserver", dest="sythilsaasserver", my_default='saas', help="The name of the saas server database")

You will also need to assign the module to be a server wide module
group.add_option("--load", dest="server_wide_modules", help="Comma-separated list of server-wide modules.", my_default='web,web_kanban,sythil_saas_server_domain_filter')

Once setup you can add domains under the saas->domains, when a person visits the domain it will do a lookup and automatically choose the appropiate database
