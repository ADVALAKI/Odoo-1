Description

Log call times
*IMPORTANT* camera access requires https for Google Chrome web browser

Instructions

1. Go to a user and hit the "Make Call" button
2. Accept access to camera/audio
3. Share URL with other user
4. Click call button

6. Inter system communication e.g. other SIP clients
