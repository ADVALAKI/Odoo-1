Description

Signs the person who fills in the form into a mailchimp list
*IMPORTANT* Updates are not synced back to Odoo so you will need to manually sync unsubscribes if also use Odoo mass mail

How to

1. Create a form in the backend via settings->HTML forms
2. Create a submit action, inputng in your Mailchimp API key
3. Select which list the new record should be subscribed to
4. Copy the generated code or use the snippet module
5. Edit the HTML of the form, adding fields with the name "email", "contact_firstname", "contact_lastname"
6. Once the form is submitted a new person is subscribed to your list with the email/first name/last name provided in the form
