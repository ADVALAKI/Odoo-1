Description

Apply site wide styles to various HTML tags
Includes changes like font-family, font-color and font-size
Also let's you override LESS variables, so you can make sweeping changes to a theme

Site wide styles

Apply site wide styles to various HTML tags.

Instructions
1. Go to Website Admin->Configuration->Style Manager
2. Select HTML tags and apply styles
3. These styles will be automatically applied to any page using the standard website layout(website.layout)
4. You can turn off custom styles via the customize menu in the website editor
